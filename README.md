# React Clima — Aplicativo de Previsão do Tempo

## Descrição do Projeto
Este projeto é uma aplicação desenvolvida em **React + Vite** que permite ao usuário consultar informações climáticas em tempo real. A aplicação consome uma API externa e exibe dados como temperatura, condição do tempo e previsão para os próximos dias.  
O objetivo é oferecer uma interface rápida, leve e intuitiva para visualizar previsões do clima.


## API Externa: OpenWeather

Este projeto utiliza a API OpenWeather para buscar informações climáticas em tempo real.
Para funcionar corretamente, é necessário possuir uma API Key válida.
A chave utilizada no código pode deixar de funcionar, caso isso aconteça você tera que entra em src/app.jsx e trocar o valor da const key,
e a chave não funciona de imediato demora 10 min até 1 hora, mas não passa de 10 min.



## Dependências
As principais dependências do projeto são:

- **React** — Biblioteca principal para construção da UI  
- **Vite** — Ferramenta de build e dev-server rápido  
- **Axios** — Cliente HTTP usado para consumir APIs  

Todas as dependências são instaladas automaticamente via `npm install`.



## Instalação 
Você deve ter nodejs instalado para funcionar o react , versão mais atualizada de preferência.
-npm install

## Para rodar o projeto
-npm run dev
